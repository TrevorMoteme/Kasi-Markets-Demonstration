# KASI Backend Portfolio

A full-stack web application demonstrating my skills in Python, FastAPI, and backend development.

## 🚀 Features

- **User Authentication:** Register, login, JWT-based sessions
- **Business Profiles:** Create and manage business listings
- **Content Feed:** Personalized feed with scoring algorithm
- **Recommendation Engine:** Location-based and preference-based scoring
